# 🏆 WorldCupAI
### AI-Powered FIFA World Cup 2026 Knockout Prediction Platform

> End-to-End Machine Learning, Deep Learning, Ensemble Learning, Monte Carlo Simulation, Explainable AI (XAI), and Interactive Streamlit Dashboard for FIFA World Cup 2026 Knockout Stage Prediction.

---

## 📌 Overview

WorldCupAI is a production-style machine learning project designed to predict the complete FIFA World Cup 2026 Knockout Stage using modern AI techniques.

The project follows an end-to-end ML lifecycle including:

- Data Engineering
- Feature Engineering
- Machine Learning
- Deep Learning
- Ensemble Learning
- Monte Carlo Simulation
- Explainable AI (XAI)
- Interactive Streamlit Dashboard

The repository is organized into multiple research-oriented phases, each validated through automated testing and detailed technical documentation.

# 📊 Dataset Summary

The project combines multiple football datasets including:

- International Football Results (1872–2026)
- FIFA Men's Rankings
- Elo Ratings
- FIFA World Cup Historical Database
- Transfermarkt Player Data
- FIFA Player Attributes
- Team Form Statistics
- Match-Level Features

Resulting in a consolidated master dataset of more than **41,000 international matches** for training and evaluation.

---

# ⚙️ Feature Engineering

The project engineers more than **72 football-specific features** across multiple categories:

- Team Strength
- Elo Ratings
- FIFA Rankings
- Recent Form
- Head-to-Head Statistics
- Attack Ratings
- Defensive Ratings
- Tournament Experience
- Context Features
- Rest Days
- Historical Performance

A feature selection pipeline reduces redundancy while preserving predictive performance.
