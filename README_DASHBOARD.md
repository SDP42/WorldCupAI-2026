# 🖥️ WorldCupAI — Dashboard Visualization Layer

## Key Interface Screens

1. **🏠 Home Page**: Platform introduction showing predicted champion, runner-up, accuracy metrics, and active voting ensemble weights.
2. **🏆 Tournament Overview**: Renders Monte Carlo team statistics, biggest upsets, and closest/most competitive matchups.
3. **🌳 Interactive Bracket**: Responsive tree-grid mapping predictions from Round of 32 down to the Final match.
4. **⚽ Match Predictions**: Searchable tabular log for all predicted matchups, with clickable details showing attributions.
5. **🤖 Ensemble Analysis**: Voting consensus levels and contribution of sub-models.
6. **📈 Explainable AI**: Plotly visualizations for local match attributions and global feature rankings.
7. **📊 Model Performance**: Sorting leaderboards, training time vs accuracy trade-offs, and ECE calibration comparison.
8. **📉 Confidence Analytics**: Distribution of match prediction certainty and margins.
9. **🔬 Counterfactual Analysis**: Scenario perturbation filter to audit prediction robustness.
10. **⭐ Feature Importance**: Categorized feature ranking with search filter and CSV downloader.
11. **📑 Reports & Downloads**: Live preview of XAI files and one-click data downloads.
12. **⚙ Settings**: Path configs and cache cleanup routines.
